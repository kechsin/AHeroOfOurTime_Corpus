Правила ввода
Поиск слова, встречающегося в любой грамматической форме
Для этого введите слово без кавычек в любой форме. Например, слово поиск найдет все предложения, где слово поиск встречается в любой грамматической форме.
Поиск определенной словоформы
Для этого введите слово в нужной форме в кавычках. Например, “поиска” найдет предложения только с этой грамматической формой.
Исключение неоднозначности слова
Для этого введите слово и через плюс (без пробелов) напишите нужную часть речи. Например, знать+NOUN найдет все предложения, где встречается существительное знать. Не используйте кавычки.
Поиск предложения, в которой есть определенные части речи или сочетания частей речи
Для этого введите часть речи или несколько частей речи через пробел. Например, NOUN VERB ADVB, ADJ дом, знать+VERB "осенью" дубом, NOUN найдет все предложения с существительными.
Особенности разметки
Жирным выделены исĸомые слова
Подчерĸнуты слова с устаревшим значением
Как обозначать часть речи
NOUN: существительное;
VERB: глагол;
ADJ: прилагательное;
ADV: наречие;
AUX: вспомогательный;
CCONJ: сочинительный союз;
NUM: цифра;
PART: частица;
PRON: местоимение;
PROPN: имя собственное;
SCONJ: подчинительный союз.
